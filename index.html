Learn German Colors with AI

This project turns a simple German-English PDF into an interactive quiz using Streamlit, PyMuPDF, and Python.

  • Upload a German-English PDF hosted on GitHub
  • Auto-extracts vocab using AI tools
  • Generates a multiple-choice quiz
  • Tracks your score and gives feedback
